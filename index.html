What the product shows

An engineer- and data-scientist-focused UI for evaluating multi-agent systems: runs, findings, metrics, and policy/judge configuration. Optionally, it can proxy to a FastAPI backend.

Runs

Review executions: inputs, tool calls, timing, and key events — with reproducible metadata.

  • Searchable history
  • Trace-style timelines
  • Export-friendly context

Findings

Track failures, near-misses, and “worked but risky” behavior with evidence.

  • Structured notes
  • Severity & tags
  • Linked to runs

Policy & judge configuration

Define evaluation criteria and keep it consistent while your system evolves.

  • Readable presets
  • Judging knobs
  • Comparable experiments
